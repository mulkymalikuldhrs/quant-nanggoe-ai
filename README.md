## ğŸ›  Setup & Installation

1. **Clone & Install**:
   ```bash
   npm install
   ```
2. **Environment Variables**:
   Konfigurasi file `.env.local` dengan kunci API Anda:
   - `GEMINI_API_KEY`: Mesin kecerdasan inti.
   - `ALPHAVANTAGE_API_KEY`: Data Forex & Makro.
   - `FINNHUB_API_KEY`: Berita & Sentimen.
   - `POLYGON_API_KEY`: Data Saham Institusional.

3. **Launch**:
   ```bash
   npm run dev
   ```
