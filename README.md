## üöÄ Key Upgrades (v12.0)

### 1. Institutional Trading Logic (SMC & S/R)
- **Smart Money Concepts (SMC)**: Deteksi otomatis Order Blocks, Fair Value Gaps (FVG), dan Market Structure Breaks (MSB).
- **Advanced S/R & Pivot**: Identifikasi area penawaran dan permintaan (Supply/Demand) berdasarkan volume dan pergerakan harga historis.
- **Retail Technicals**: Integrasi strategi klasik (Golden Cross, RSI Divergence) sebagai filter konfirmasi.

### 2. MT4/MT5 Professional Portfolio
- **Terminal Interface**: Portfolio management kini menyerupai terminal MT4/MT5 dengan metrik real-time: Balance, Equity, Margin, Free Margin, dan Margin Level.
- **Real-Time PnL**: Perhitungan profit/loss yang akurat berdasarkan data market terkini.
- **Asset Command**: Kontrol terpusat untuk semua posisi trading di berbagai instrumen (Crypto, Forex, Saham).

### 3. Professional UI & Window System
- **Resizable & Draggable**: Semua jendela sistem (Market, Terminal, Browser) kini sepenuhnya dapat diubah ukurannya dan digeser untuk pengaturan workspace yang personal.
- **PC-Optimized Layout**: Antarmuka yang disesuaikan secara dinamis agar pas di monitor layar lebar tanpa distorsi atau elemen yang terlalu besar.
- **Branding Cleanup**: Mengganti istilah "Neural Chart" menjadi "Market Analysis" yang lebih profesional dan menghapus noise visual yang tidak perlu.

### 4. Autonomous Research & Intelligence
- **Real Research Agent**: Agen riset kini bekerja secara otonom dengan akses memori persisten untuk menganalisis tren makro dan mikro secara mendalam.
- **Neural Browser with Reader**: Browser terintegrasi kini dilengkapi dengan *Neural Reader* (via Jina AI) untuk mengekstrak informasi dari situs web yang terproteksi.
- **Fix `/scan` Command**: Perintah `/scan [SYMBOL]` memberikan output analisis teknikal asli, sinyal SMC, dan skor konsensus pasar secara instan.

### 5. Robust Data Pipeline
- **Institutional API Access**: Menggunakan sistem proxy rotator canggih untuk menjamin aliran data dari Binance, Polygon, dan sumber lainnya tanpa hambatan CORS.
- **Persistent Memory**: Memori agen yang ditingkatkan untuk menyimpan konteks antar sesi kerja, memungkinkan kolaborasi antar-agen yang lebih koheren.

## üõ† Setup & Installation

1. **Clone & Install**:
   ```bash
   npm install
   ```
2. **Environment Variables**:
   Konfigurasi file `.env.local` dengan kunci API Anda:
   - `GEMINI_API_KEY`: Mesin kecerdasan inti (Nanggroe Pro 12.0).
   - `ALPHAVANTAGE_API_KEY`: Data Forex & Makro.
   - `FINNHUB_API_KEY`: Berita & Sentimen.
   - `POLYGON_API_KEY`: Data Saham Institusional.

3. **Launch**:
   ```bash
   npm run dev
   ```
