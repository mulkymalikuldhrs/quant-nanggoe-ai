1. PENDAHULUAN: APA ITU QUANT NANGGROE?
---------------------------------------
Quant Nanggroe AI adalah Sistem Operasi Trading Kuantitatif (Trading OS) yang menggabungkan antarmuka macOS Glass Edition dengan kecerdasan buatan tingkat lanjut (Artificial Intelligence). 

Bukan sekadar bot trading, ini adalah ekosistem "Bloomberg Killer" yang dirancang untuk memberikan trader retail alat yang sama canggihnya dengan institusi besar (Hedge Funds).

2. ARSITEKTUR MULTI-AGENT (NEURAL SWARM)
----------------------------------------
Sistem ini bekerja berdasarkan konsep "Neural Swarm Parallelism". Artinya, alih-alih hanya satu otak AI yang bekerja, Quant Nanggroe mengaktifkan 5 agen spesialis secara SIMULTAN untuk setiap perintah kompleks:

A. Alpha Prime (The Coordinator)
   - Otak pusat yang mengarahkan semua agen.
   - Bertugas mensintesis hasil analisis menjadi satu rencana aksi final.
   - Mengontrol interface (buka window, navigasi browser, dll).

B. Quant-Scanner (Technical Specialist)
   - Fokus pada data numerik dan indikator teknikal.
   - Menganalisis RSI, MACD, Volume Profile, dan Order Flow secara real-time.

C. News-Sentinel (Macro & Sentiment Specialist)
   - Menggunakan Autonomous Browser untuk mencari berita global.
   - Menganalisis sentimen pasar dari media sosial dan rilis data ekonomi (Fed, CPI, dll).

D. Risk-Guardian (Security & Risk Specialist)
   - Menghitung rasio Risk/Reward.
   - Memastikan setiap rencana trading mematuhi protokol keamanan modal (Stop Loss/Take Profit).

E. Strategy-Weaver (Algo Development)
   - Menulis dan mengoptimalkan algoritma trading menggunakan Python.
   - Melakukan backtesting terhadap strategi yang diusulkan.

3. ALUR KERJA SISTEM (STEP-BY-STEP FLOW)
----------------------------------------
Step 1: INPUT
   User memberikan perintah via "Neural Terminal" atau "OmniBar" (Shortcut: Klik icon Search).
   
Step 2: SWARM TRIGGER
   Kernel mendeteksi jika perintah membutuhkan analisis mendalam. Jika iya, mode "Parallel Spawning" diaktifkan.

Step 3: PARALLEL EXECUTION
   Kelima agen "terbang" secara bersamaan. Anda bisa melihat log aktivitas mereka di Swarm Monitor (Icon Bot di Dock).

Step 4: NEURAL SYNTHESIS
   Alpha Prime mengumpulkan data dari semua agen. Jika ada perbedaan pendapat (misal: Quant bilang Buy, News bilang Sell), Alpha Prime akan melakukan rekonsiliasi data berdasarkan logika institusional.

Step 5: AUTONOMOUS ACTION
   Sistem tidak hanya menjawab dengan teks, tapi juga melakukan aksi nyata:
   - Membuka Chart di Market Data window.
   - Membuka berita relevan di Neural Browser.
   - Mempersiapkan order di Trading Terminal.

4. FITUR UNGGULAN YANG SUDAH TERSEDIA
-------------------------------------
- Real-Time Binance Integration: Data harga diambil langsung dari Binance (bukan simulasi).
- Neural Memory: AI mengingat strategi Anda, gaya trading Anda, dan percakapan sebelumnya.
- Swarm Graph: Visualisasi koneksi antar-agen yang aktif secara real-time.
- Adaptive Glass UI: Interface transparan yang menyesuaikan ukuran layar secara otomatis.
- Visual Guide: Jendela khusus yang menjelaskan arsitektur sistem secara grafis.

5. CARA MENGAKSES VISUALISASI (VISUAL GUIDE)
--------------------------------------------
Untuk melihat diagram arsitektur dan spesialisasi agen secara visual:
1. Arahkan mouse ke Dock (Taskbar bagian bawah).
2. Cari icon otak (Brain Icon) berwarna hijau.
3. Klik "Visual Guide".
4. Jendela "System Architecture" akan terbuka, menampilkan diagram alur kerja A-Z.

6. TEKNOLOGI YANG DIGUNAKAN
---------------------------
- Frontend: React + Vite + Tailwind CSS (Glassmorphism).
- Intelligence: Google Gemini 2.0 Flash (sebagai Neural Core).
- Data API: Binance Institutional API (Rest/Websockets).
- State Management: React Adaptive Context dengan Neural Memory persistence.
