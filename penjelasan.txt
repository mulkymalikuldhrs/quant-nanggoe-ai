2. BAGAIMANA MULTI-AGENT BEKERJA?
Sistem ini menggunakan arsitektur "Neural Swarm Parallelism" di mana lima agen spesialis bekerja secara simultan:
   - Alpha Prime (Lead): Komandan utama yang mensintesis data dan mengambil keputusan final.
   - Quant-Scanner (Technical): Spesialis indikator teknikal (RSI, MACD, Bollinger, dll).
   - News-Sentinel (Macro/Fundamental): Memantau berita global, sentimen pasar, dan data makro.
   - Risk-Guardian (Risk Management): Penjaga keamanan modal, menghitung rasio risk/reward secara real-time.
   - Strategy-Weaver (Algo Dev): Pengembang algoritma yang terus mengoptimalkan logika trading.

Saat Anda memberikan perintah (misal: "Analisis BTC"), sistem tidak hanya memberikan satu jawaban LLM biasa. Sebaliknya, semua agen ini "terbang" (spawning) secara paralel, menganalisis dari sudut pandang masing-masing, dan hasilnya disatukan oleh Alpha Prime menjadi satu rencana aksi yang matang.

4. ALUR KERJA (SYSTEM FLOW):
   Step 1: Input -> User memberikan perintah via Neural Terminal atau OmniBar.
   Step 2: Processing -> Alpha Prime menentukan apakah tugas membutuhkan Swarm Intelligence.
   Step 3: Execution -> Jika ya, Agen spesialis diaktifkan secara paralel (Parallel Spawning).
   Step 4: Synthesis -> Alpha Prime mengumpulkan semua analisis dan menyusun System Action.
   Step 5: Action -> Sistem secara otomatis membuka window yang diperlukan (Browser, Chart, Terminal) atau melakukan eksekusi trading.
   Step 6: Learning -> Hasil disimpan dalam Neural Memory untuk optimasi di masa depan.
